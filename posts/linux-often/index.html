自用Linux常用命令

Linux常用命令

1.连接

1

ssh root@ip

2.文件上传下载

windows中使用xshell(强烈推荐)或者FinalShell，Mac也可以使用FinalShell。

或者使用命令：

文件上传：

1

scp 本地文件路径 用户名@ip地址:服务器要存放的的路径

例如将文件/Users/dpf/Desktop/1.txt上传到服务器122.51.114.41的根目录

1

scp /Users/dpf/Desktop/1.txt root@122.51.114.41:/

文件下载：

1

scp 用户名@IP地址:需要下载的文件路径 本地存放下载的路径

例如将122.51.114.41根目录的文件下载到本地Users路径下

1

scp root@122.51.114.41:/1.txt /Users

3.端口号相关

1.列出所有已经使用的端口号

1

netstat -ntlp

2.查看某个端口号占用情况

1
2

lsof -i:端口号
netstat -lnpt |grep 端口号

若-bash: lsof: 未找到命令，通过命令安装yum install lsof

3.关闭已经占用的端口号

1

kill -9 pid

上面第2点可以查看端口号对应pid

4.防火墙相关

Centos6.x

1
2
3
4
5
6

service iptables start         
service iptables restart

/sbin/iptables -I INPUT -p tcp --dport 8080 -j ACCEPT   	-- 开启指定端口
/etc/init.d/iptables save   																 -- 保存修改
/sbin/iptables -I INPUT -p tcp --dport 8080 -j DROP   		-- 关闭指定端口

Centos7.x

1
2
3
4
5
6
7
8

systemctl start firewalld：开启防火墙
systemctl status firewalld
systemctl stop firewalld
firewall-cmd --add-port=8080/tcp --permanent：开放8080端口
firewall-cmd --reload：重新加载(修改端口号后需要重新加载)
firewall-cmd --query-port=123/tcp：查询指定端口是否开启成功
firewall-cmd --permanent --remove-port=123/tcp：关闭指定端口
firewall-cmd --zone=public --list-ports:查看开放的所有端口

5.网络相关

查看本机公网信息

1

curl cip.cc

请求web服务器(多用于查看某个应用是否部署成功)

1
2

curl ip
curl http://127.0.0.1:8080

查看网络情况

1

ifconfig

测试网络是否连通

1

ping ip地址或域名

6.文件相关

创建文件

1
2
3
4

mkdir 目录
     -p       创建目录，若无父目录，则创建p(parent)
touch         创建空文件

编辑文件

1
2
3
4
5
6

vim / vi  文件名
     insert   编辑
     :q       退出
     :q!      强制退出
     :wq      保存并退出
     esc      退出编辑

复制文件

1
2
3

cp 需要复制的文件  需要复制到的目录[新名字]

cp 1.txt 2.txt

删除文件

1
2
3

rm 文件名
     -r       递归删除，可删除子目录及文件
     -f       强制删除

移动文件

1

mv 文件 目标目录

解压文件

  • .zip unzip xxx.zip
  • .xz xz -d xxx.tar.xz
  • .tar tar -xvf xxx.tar
  • .tar.gz tar -zxvf xxx.tar.gz

查看文件内容

echo ''    						-- 输出字符串

查看日志

4
